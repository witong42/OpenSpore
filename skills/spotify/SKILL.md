---
name: spotify
description: Control Spotify playback (play, pause, next, prev, play_song)
---

This skill allows the agent to control Spotify on macOS using AppleScript.

### Usage
- `[SPOTIFY: play]`: Resume playback.
- `[SPOTIFY: pause]`: Pause playback.
- `[SPOTIFY: next]`: Skip to next track.
- `[SPOTIFY: prev]`: Go to previous track.
- `[SPOTIFY: play_song "Song Name"]`: Search and play a specific song.
